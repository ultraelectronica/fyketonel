# Fyke's Laboratory

My portfolio. Gaming-themed, retro arcade aesthetic. You're looking at it.

## Stack

- Next.js 16 + React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Radix UI (8-bit styled components)
- Resend (contact form)

## Run It

```bash
bun dev
```

Open http://localhost:3000.

## What's Here

- RPG-style player profile with stats
- GitHub contributions display
- Interactive skill tree (Frontend/Backend/DevOps/Design branches)
- 12 projects sorted by tier (S to D)
- Playable arcade hockey mini-game
- Contact form
- Dark mode with "Ally" theme variant
